## Keyboard Shortcuts

### View Modes
| Shortcut | Action |
|----------|--------|
| `Alt + 1` | Editor only |
| `Alt + 2` | Side-by-side (editor + preview) |
| `Alt + 3` | Preview only |
| `Ctrl + E` | Toggle preview |

### Navigation
| Shortcut | Action |
|----------|--------|
| `Tab` | Cycle focus: Notebooks → Notes → Editor |
| `Shift + Tab` | Cycle focus (reverse) |
| `Ctrl + P` | Quick open (search notes) |
| `Ctrl + B` | Toggle sidebar |
| `↑ / ↓` | Navigate lists |
| `j / k` | Navigate lists (Vim mode) |
| `Enter` | Open selected note in editor |

### Notes & Notebooks
| Shortcut | Action |
|----------|--------|
| `Ctrl + N` | New note / notebook / tag (context-aware) |
| `Ctrl + Shift + N` | New sub-notebook |
| `Ctrl + W` | Close current note |
| `Delete` | Delete selected item |
| `F2` | Rename selected item |
| `A` | Create new item (when focused on list) |

### Editor
| Shortcut | Action |
|----------|--------|
| `Ctrl + +` | Zoom in |
| `Ctrl + -` | Zoom out |
| `Ctrl + 0` | Reset zoom |
| `Ctrl + Alt + V` | Toggle Vim mode |
